Food database (USDA FDC)

SR Legacy loads automatically on first visit. Branded import is optional (~large); until it finishes, barcode scans use USDA API online.

Used for barcode lookups when the offline branded database is not imported. The free DEMO_KEY allows only 30 lookups/hour; a personal key allows 1,000/hour.

Controls which online database is queried when a barcode is not in the local branded DB. Open Food Facts requires no API key and has broader international coverage. USDA has higher-quality US data but the free DEMO_KEY allows only 30 lookups per hour.

Performance Settings

When enabled, the app will use cached data instead of fetching from the internet, making it faster but potentially showing older content.
